# 第5章：工程领域应用案例 / Chapter 5: Engineering Domain Application Cases

## 本章内容

本章展示工程本体论在四大工程领域的实际应用案例：

1. **智能制造调度** - 基于本体的设备调度与冲突检测
2. **自动驾驶场景** - 交通场景语义理解
3. **建筑信息模型（BIM）** - IFC本体与合规性检查
4. **航空航天FMEA** - 故障模式与影响分析
