# 第2章：本体论基础理论 / Chapter 2: Ontology Foundations

## 本章内容

本章介绍本体论的核心组成要素（类、属性、公理、实例）、理论基础（描述逻辑、一阶逻辑）
以及工程领域本体建模的基本概念。

## 关键概念

- **类 (Class)**: 本体的基础单元，代表领域内的一类事物
- **属性 (Property)**: 数据属性（Data Property）与对象属性（Object Property）
- **公理 (Axiom)**: 约束与规则，如子类关系（⊑）、等价类（≡）
- **实例 (Individual)**: 类的具体实例，即ABox中的数据

## 符号说明

| 符号 | 含义 |
|------|------|
| ⊑ | 子类关系（SubClassOf） |
| ≡ | 等价类（EquivalentTo） |
| ∀ | 全称量词（Universal Restriction） |
| ∃ | 存在量词（Existential Restriction） |
| ∧ | 逻辑与（AND） |
| ∨ | 逻辑或（OR） |
| ¬ | 逻辑非（NOT） |
| → | 蕴含（Implies） |
| ⊓ | 交（Intersection） |
| ⊔ | 并（Union） |
